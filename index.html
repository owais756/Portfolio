Semester Success Portal

The Semester Success Portal is a dedicated academic assistance platform built using HTML and CSS, designed to help engineering students quickly access semester-wise study materials, notes, and PYQs. The interface is clean, student-friendly, and optimized for fast navigation, with well-structured subject sections that make content easy to find. The platform features a fully responsive layout with calm color themes and balanced typography for a comfortable reading experience. Deployed through GitHub Pages, the portal provides instant access anytime, helping students save time during exams by centralizing essential resources in one simple and organized hub.

Tech Stack: HTML, CSS
